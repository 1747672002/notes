绘制一个六角形

```python
#-*- coding:utf-8 -*-

import turtle as tur

tur.setup(500, 500, 200, 200)

tur.seth(90)

tur.fd(120)

tur.seth(-30)

tur.fd(120)

tur.seth(-150)

tur.fd(120)

tur.seth(30)

tur.fd(40)

tur.seth(-30)

tur.fd(40)

tur.seth(90)

tur.fd(120)

tur.seth(210)
```
